Início Ionic Tutorial Ionic 1° Passo - é abrir a pasta do ionic no cmd com: cd /ionic 2° Passo - com: "ionic start "nome"App "template" --type=angular - criar um novo projeto (após um tempo carregando, aparecerá a pergunta "create free Ionic acount (y/n") a qual você responderá não ou seja n ou no) 3° Passo - abrir o projeto no VSCode com: code . (caso queira colocar a sua conta do GitHub no projeto e salva-lo automaticamente use git config –global user.email “your@example.com ou git config –global user.name “Your Name”) 4° Passo - após aberto seu projeto poderá começar a trabalhar livremente, não se esqueça de abrir a página do Ionic Framework e entrar na página UI Components, para colocar os comandos basta adicionalos no seu código html e colocalo entre a tag ion-content Aviso : Você deve ter baixado o nodeJS LTS
